如何搭建 GitHub Pages 个人网站

2026-05-01 教程, GitHub

本文将介绍如何从零开始搭建一个基于 GitHub Pages 的个人学术主页。

1. 创建 GitHub 仓库

首先，在 GitHub 上创建一个名为 weijiansh1.github.io 的仓库，其中 weijiansh1 替换为你的 GitHub 用户名。

2. 上传网站文件

将你的网站文件推送到该仓库：

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/weijiansh1/weijiansh1.github.io.git
git push -u origin main

3. 配置 GitHub Pages

进入仓库的 Settings > Pages，选择 main 分支作为发布源。几分钟后，你的网站就可以通过 https://weijiansh1.github.io 访问了。

4. 自定义域名（可选）

如果你有自己的域名，可以在 Settings > Pages 中配置自定义域名，并在域名服务商处添加 CNAME 记录。

DNS 配置示例

类型: CNAME
名称: www
值: weijiansh1.github.io

总结

GitHub Pages 提供了一个免费、简单的方式来托管个人网站。配合纯 HTML/CSS/JS，你可以完全控制网站的每一个细节。
